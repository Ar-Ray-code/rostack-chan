# rostack-chan
stack-chan hardware using ros2_control




## Requirements

- ROS 2 Humble
- [feetech_scs_ros2_driver](https://github.com/Ar-Ray-code/feetech_scs_ros2_driver)

## Usage

```bash
ros2 launch rostackchan_example rostackchan_random.launch.py
```
